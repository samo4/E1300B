D-SCPI Drivers

This page contains information about the Agilent SCPI (Standard Commands for Programmable Instruments) Instrument Drivers to download into Agilent VXI Command Modules (E1300, E1301, E1306, E1405, E1406).

  • Use the .dc files to download into the VXI Command Module via RS-232 Serial.
  • Use the .du files to download into the VXI Command Module via GPIB.

This page contains links to:

  • Supported Drivers
  • VTI Instruments Drivers
  • Programs to Download SCPI Drivers

Use your browser's Find feature (Ctrl-F) to quickly locate an instrument's model number below.

Programs to Download SCPI Drivers

VEE Program to Download SCPI Drivers via GPIB

vxidld.vee
  • Removes all of the currently loaded drivers.
  • Destroys all contents in the current RAM Disk (if any).
  • Downloads the drivers that you entered by filename in the box "Files".

HP VIC to Download Drivers

The Hewlett-Packard VXI Installation Consultant (HP VIC) is a VXI hardware installation program to help you configure and install VXI instruments and can also download DOS-formatted instrument drivers into VXI Command Modules.

vic32setup.exe: VIC 3.1 is a 32-bit Windows program for Windows 95 and Windows NT.
  1. Create a temporary directory on your PC hard disk, and download the archive file into this temporary directory.
  2. Run that (WinZip) archive file:
    • Fill in a destination folder: the A: floppy or an empty hard disk directory.
    • Click the Unzip button to explode the files.
    • Click the Close button.
  3. Run the new Setup.exe program to continue the installation.

Instructions for using HP VIC and its Driver Download capability are contained in the software's online help.

Download the .dc and .du SCPI driver files into the DRIVERS directory under the directory where you installed HP VIC. This is where HP VIC will look for SCPI drivers to download to the command module during its configuration process. If it cannot find them there, it will ask you for each driver and expect it to be on a floppy disk.

HP VIC supports an Agilent 24542U serial cable (9F/9F) between the PC's COM1 port and the Command Module.

DOS Program to Download SCPI Drivers via Serial

VXIDLD.EXE works well on DOS 6.22, or a in a DOS window on Windows 3.11, Windows 95, Windows NT 3.51, or Windows NT 4.0.
  1. Attach an Agilent 24542U serial cable (9F/9F) between the PC's COM1 port and the Command Module.
  2. Copy VXIDLD.EXE, VXIDLD.CFG, and your *.dc files into an empty directory.
  3. Optional: Edit the VXIDLD.CFG text file to use a different PC COM port (line 390), or to delete previously downloaded SCPI drivers (line 504).
  4. Execute the VXIDLD.EXE program and monitor the progress on the screen. (The 133K byte DIG_IO.DC file takes about 4 minutes to download.) Execute the VXIDLD.EXE program from an MS-DOS Prompt window so you can see any error messages.

HP BASIC Workstation Program to Download SCPI Drivers

On your HP BASIC Workstation (Series 200/300) use the GET command to read the text-based VXIDLD.GET program:

GET "VXIDLD.GET"
